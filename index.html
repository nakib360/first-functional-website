Important FAQ

What is the documant objact model?

The Document Object Model (DOM) represents HTML or XML documents as a tree structure, allowing programmatic manipulation of content.

How do you select an element from DOM?

I use document.querySelector() for a single element or document.querySelectorAll() for multiple. Alternatively, use getElementById() or getElementsByClassName().

What is event deligation in the context of the DOM,and why is it useful?

Event delegation allows a parent element to handle events for its children using event bubbling. It's useful for performance optimization, managing dynamically added elements, and reducing multiple event listeners in the DOM.

How do you manipulate an element's attributes and styles using the DOM?

I use setAttribute(), getAttribute(), and removeAttribute() for attributes. Modify styles with element.style.property or manage classes using classList.add(), remove(), and toggle() for efficient DOM manipulation.
